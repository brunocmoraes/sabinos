# Pizza links

## Start

Install dependencies:

```sh
yarn

# With npm
npm install
```

4. Start the development server:

```sh
yarn dev

# With npm
npm run dev
```

5. Open your browser and visit [http://localhost:5173](http://localhost:5173) to see your project.
